# 🎬 Onebite Cinema - App Router Ver

한입 챌린지 8기 **Next.js 4주차 프로젝트**  
영화 데이터를 조회하고 검색할 수 있는 **영화 정보 웹 서비스**입니다.

한입 씨네마에 **스트리밍, 서버액션, 병렬 라우팅 기능**을 적용합니다.

프론트엔드는 **Next.js**로 구현되었으며, 백엔드 서버와 연동하여 실제 데이터를 기반으로 동작합니다.

## 🚀 배포 링크 (Vercel)

http://onebite-movie-mission-app-router-fr.vercel.app/

## 📌 주요 기능

### 1) 스트리밍 적용

- **인덱스 페이지**와 **서치 페이지**에 각각 Suspense를 활용해 **스트리밍**을 적용

### 2) 서버 액션으로 리뷰 기능 추가

- **리뷰 조회**: 백엔드 서버의 `~/review/movie/[id]` API를 통해 리뷰 데이터를 불러와 렌더링
- **리뷰 추가**: 서버 액션을 활용해 사용자가 입력한 리뷰를 데이터베이스에 추가하고, `revalidateTag`로 화면에 즉시 반영
- **리뷰 삭제**: 삭제 버튼 클릭 시 서버 액션으로 리뷰를 삭제하고, `revalidateTag`로 화면에서 즉시 제거

### 3) 병렬 라우팅을 활용한 인터셉팅 모달

- **CSR 방식 접근 시**: 영화 상세 페이지(`/movie/[id]`)가 모달 창으로 열림
- **주소 직접 입력 · 새로고침 시**: 전체 페이지로 열림
